Aplikasi CRUD (Create, Read, Update, Delete) menggunakan PHP, MySQL, dan Bootstrap.

## Cara Menjalankan

2. Letakkan folder project di www/ (Laragon)
3. Buka browser ke http://localhost/T4-week10/
